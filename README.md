# Social Farm AI OS

An intelligent content operations platform that transforms ideas into high-quality multimedia content through coordinated AI agents, structured workflows, and human oversight.

## Project Structure

- `docs/`: Project specifications, prompts, and architecture diagrams.
- `frontend/`: Next.js application.
- `backend/`: FastAPI application.
- `assets/`: Global project assets.
- `scripts/`: Automation and utility scripts.
- `infrastructure/`: Infrastructure configurations.
- `docker/`: Docker containerization.

## Getting Started

1. Ensure Docker and Node.js (v20+) are installed.
2. See individual service directories for setup instructions.
