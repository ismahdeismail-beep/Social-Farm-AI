## [0.3.0] - 2026-06-26
### Added
- **Content Strategy Agent (Phase P11)**
  - 20 database models for content strategy system
  - Strategy Engine with comprehensive strategy generation
  - Opportunity Engine for content opportunity detection
  - Campaign Planner for campaign creation and management
  - Calendar Generator for content scheduling
  - Audience Analyzer for audience segmentation
  - Competitor Analyzer for competitive analysis
  - Forecast Engine for performance predictions
  - Recommendation Engine for AI-powered suggestions
  - 12 REST API endpoints (/strategy/generate, /strategy, /strategy/calendar, etc.)
  - 4 frontend pages (Strategy Dashboard, Campaign Planner, Content Calendar, Opportunity Board)
  - Zustand store for state management
  - Comprehensive test suite

## [0.2.0] - 2026-06-26
### Added
- **Master AI Orchestrator (Phase P9)**
  - 25+ database models for AI system
  - AI Gateway with multi-provider support
  - Model Router with intelligent routing
  - Memory Engine for context persistence
  - Prompt Library with version control
  - Agent Registry for agent management
  - Workflow Engine for complex tasks
  - Quality Engine for output validation
  - 11 REST API endpoints
  - 4 frontend pages (Command Center, Agents, Models, Health)
  - Comprehensive test suite
