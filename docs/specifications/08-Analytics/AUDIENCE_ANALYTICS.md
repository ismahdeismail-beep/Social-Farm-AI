# AUDIENCE_ANALYTICS

## Purpose
Audience Analytics provides insights into *who* is engaging with the content, enabling hyper-targeted strategies.

## Data Dimensions
- **Demographics:** Age, gender, interests (derived where permitted).
- **Geography:** Regional concentration of audience.
- **Technology:** Device types, browser usage, platform versions.
- **Behavioral:** Peak activity times, content preferences.

## Usage
Audience data is used to optimize posting times (via [SCHEDULER](../07-Publishing/SCHEDULER.md)) and tailor content tone for specific audience segments.
