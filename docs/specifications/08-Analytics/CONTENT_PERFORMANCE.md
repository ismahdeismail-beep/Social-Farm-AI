# CONTENT_PERFORMANCE

## Purpose
Content Performance analytics measure the effectiveness of published content across platforms, guiding future creation strategies.

## Key Metrics
| Metric | Definition |
| :--- | :--- |
| **Reach** | Total unique users who saw the content |
| **Views/Impressions** | Total times the content was displayed |
| **Watch Time** | Total time users spent watching (for video) |
| **Retention** | Average duration of content viewed |
| **Virality Index** | Ratio of Shares to Views |

## Analytics Feedback Loop
Content performance data is aggregated and sent back to the [AI_RECOMMENDATIONS](./AI_RECOMMENDATIONS.md) service, which refines future content generation prompts.
