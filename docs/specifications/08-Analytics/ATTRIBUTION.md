# ATTRIBUTION

## Purpose
Attribution models identify which content or campaigns drive conversions, allowing for precise ROI calculation.

## Attribution Models Supported
- **First-Click:** Credits the first content a user engaged with.
- **Last-Click:** Credits the final content before conversion.
- **Linear:** Equally credits all touchpoints in the journey.
- **Time-Decay:** Credits touchpoints closer in time to the conversion more heavily.

## Implementation
Attribution data is tracked using UTM parameters and tracking pixels associated with external link clicks within published posts, integrated with the platform's [DATA_PIPELINE](./DATA_PIPELINE.md).
