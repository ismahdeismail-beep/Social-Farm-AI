# VISUALIZATION_GUIDELINES

## Purpose
These guidelines ensure consistency, accessibility, and clarity in all data visualizations across the platform.

## Design Principles
- **Chart Selection:** Choose the simplest chart that conveys the insight (e.g., Bar for comparison, Line for trends).
- **Color Palettes:** Use accessibility-compliant color schemes, avoiding reliance on color alone to convey meaning.
- **Labeling:** Clear, concise labeling of axes and legends.
- **Accessibility:** All charts must provide alternative text or data table views.
