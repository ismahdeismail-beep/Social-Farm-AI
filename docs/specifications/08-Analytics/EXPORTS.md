# EXPORTS

## Purpose
The Export module provides flexible data extraction for users requiring offline analysis or integration with external BI tools.

## Supported Formats
- **CSV/Excel:** For spreadsheet analysis.
- **PDF:** For formatted reports.
- **API (JSON):** For automated data consumption by external applications.

## Mechanisms
- **Direct Export:** Immediate download from UI.
- **Scheduled Exports:** Automated delivery via email or SFTP/Webhook.
