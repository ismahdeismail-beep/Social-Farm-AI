# SECURITY

## Purpose
Analytics security protects PII, sensitive organizational data, and system integrity.

## Core Security Controls
- **PII Protection:** Data masking and anonymization at the point of ingestion.
- **Workspace Isolation:** Strict data partitioning; workspaces can only access their own data.
- **RBAC:** Fine-grained access control to dashboards, reports, and raw data.
- **Encryption:** Encryption at rest and in transit.
- **Audit Logging:** Immutability for all access logs, tracking who accessed what and when.
