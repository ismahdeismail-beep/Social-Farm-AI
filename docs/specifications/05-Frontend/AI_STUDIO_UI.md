# Frontend Specification: AI_STUDIO_UI

## Purpose
The core workspace for interacting with AI models and agents.

## Design
- **Left Panel:** Chat/Prompt history and Agent selector.
- **Center Panel:** Main chat/prompt editor and response viewer.
- **Right Panel:** Context/Memory viewer and parameters.

## Functionality
- Structured Prompt Editor.
- Response stream visualization.
- Human-in-the-loop approval mechanism.
