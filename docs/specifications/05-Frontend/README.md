# Frontend Specification: README

## Overview
This directory contains the comprehensive frontend architectural specifications for Social Farm AI OS. The architecture is based on Next.js 14+ (App Router), React, TypeScript, and Tailwind CSS.

## Implementation Order
1. **Foundation:** `FRONTEND_OVERVIEW.md`, `APPLICATION_STRUCTURE.md`, `ROUTING.md`, `THEMING.md`, `LAYOUT_SYSTEM.md`.
2. **Core Services:** `STATE_MANAGEMENT.md`, `API_CLIENT.md`.
3. **Core Components:** `COMPONENT_LIBRARY.md`, `FORM_SYSTEM.md`, `TABLE_SYSTEM.md`, `MODAL_SYSTEM.md`, `NOTIFICATION_SYSTEM.md`.
4. **Features (P1-P2):** `AUTHENTICATION_UI.md`, `DASHBOARD_UI.md`, `BRAND_MANAGER_UI.md`.
5. **Features (P3-P8):** `AI_STUDIO_UI.md`, `SCRIPT_STUDIO_UI.md`, `MEDIA_FACTORY_UI.md`, etc.
6. **Quality & Ops:** `ACCESSIBILITY_IMPLEMENTATION.md`, `PERFORMANCE.md`, `TESTING_STRATEGY.md`.
