# Frontend Specification: PROJECT_MANAGER_UI

## Purpose
Manage content projects and campaign timelines.

## Responsibilities
- Project creation and organization.
- Campaign milestone tracking.
- Task assignment and tracking.
- Calendar view for project planning.

## Architecture
- Task board (Kanban) or Table view.
- Timeline (Gantt) visualization.

## User Interactions
- Drag-and-drop task movement.
- Milestone deadline setting.
