# Metrics

## Key Operational Metrics
*   **Availability**: Percentage of time services are up.
*   **Latency**: P95/P99 response times.
*   **Queue Length**: Number of pending AI tasks.
*   **AI Response Time**: Time taken for AI inference.
*   **CPU/Memory**: Resource utilization.
*   **Storage**: Disk usage.
*   **User Activity**: Active users, request volume.
*   **Publishing Success**: Success rate of publishing tasks.
