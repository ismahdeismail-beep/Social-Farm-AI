# Backup Strategy

## Backup Types
*   **Daily**: Full backup of databases and object storage.
*   **Incremental**: Hourly incremental backups for critical databases.

## Verification
Automated restoration tests performed monthly to ensure backup integrity.

## Retention
*   **Daily Backups**: Retained for 30 days.
*   **Weekly Backups**: Retained for 90 days.
*   **Monthly Backups**: Retained for 1 year.
