# Purpose

The Core module defines the strategic, architectural, and engineering foundations of Social Farm AI OS.

Every specification in this repository depends on the documents contained within this module.

Before implementing any feature, AI agent, workflow, user interface, or infrastructure component, this module must be read and understood.

# Scope

This module establishes:

- Project vision
- Software architecture
- Technology stack
- Engineering standards
- Development methodology
- File organization
- Terminology
- Long-term roadmap

These documents serve as the single source of truth for the entire project.
