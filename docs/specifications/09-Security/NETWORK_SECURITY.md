# Network Security

## 1. Purpose
To protect the network infrastructure from unauthorized access and attacks.

## 2. Architecture
Multi-layered network security with VPC, subnets, and firewalls.

## 3. Responsibilities
*   **DevOps/Network Engineer**: Configure network security policies.

## 4. Threats
*   DDoS attacks.
*   Unauthorized network access.
*   Man-in-the-middle attacks.

## 5. Mitigations
*   Firewalls (Security Groups).
*   Reverse proxy (Nginx).
*   TLS encryption.

## 6. Best Practices
*   Use private subnets for all services.
*   Implement network segmentation.

## 7. Security Controls
*   Network ACLs.
*   Security Groups.

## 8. Monitoring
*   Monitor network traffic logs.

## 9. Incident Handling
*   Block malicious IP ranges.

## 10. Future Enhancements
*   Intrusion Detection System (IDS).
