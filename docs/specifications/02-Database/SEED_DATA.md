# SEED DATA

This document defines the base data required for system initialization.

## Roles & Permissions
- `owner`: Full system access.
- `admin`: Operational access (Users, Config).
- `manager`: Campaign management.
- `editor`: Content creation/editing.
- `publisher`: Scheduling and publishing permissions.

## Default Workspace/Org
- Initial system initialization should create:
    - Organization: "Global Admin Org"
    - Workspace: "Default Sandbox"
    - Admin User: "system_admin"

## Development Data
- Pre-populated prompt templates for: `script_writing`, `trend_analysis`, `image_generation`.
